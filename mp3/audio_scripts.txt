===============================================================================
  INTELLIGENT MULTI-GAS SAFETY SYSTEM — AUDIO ALERT SCRIPTS
===============================================================================

File naming convention: DFPlayer Mini requires MP3 files named 0001.mp3,
0002.mp3, etc., placed in the root directory of the MicroSD card in mp3 folder - (FAT32).

Recording specs:
  - Format      : MP3, 128 kbps, mono, 44.1 kHz
  - Peak volume : Normalized to -3 dB
  - Pad         : 300 ms silence at start of each file (prevents clipping)
